### 🚧 安装方式
* 基础依赖项目: 
```lua 
fly-framework
```
* IDEA 打开 fly-framework
```lua 
cd /fly-dependencies
mvn clean 
mvn install
```
* 返回根目录安装基础依赖(fly-framework)
```lua 
cd ..
mvn clean 
mvn install
```
